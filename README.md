# react-sass-webpack-boilerplate
> React boilerplate, using [Sass](http://sass-lang.com/) imported from jsx, [react-router](https://github.com/rackt/react-router), [Bootstrap](http://getbootstrap.com/) and [Webpack](https://webpack.github.io/)

##Prerequisites
You will need to have [git](https://git-scm.com/downloads) and [npm](https://nodejs.org/en/) installed.
[Git bash](https://git-for-windows.github.io/) is recommended for Windows

##Installation
Clone the repository, e.g.
```sh
$ git clone https://github.com/lukead/react-sass-webpack-boilerplate.git
```

Install dependencies
```sh
$ npm install
```
*n.b. you may receive errors relating to python. The install should continue without this.*

##Running in development mode
Start the webpack development server
```sh
$ npm start
```

Navigate to http://localhost:8080/
Webpack will automatically reload your updates.

##Production builds
```sh
$ npm run build
```
